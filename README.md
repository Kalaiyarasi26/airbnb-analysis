# Airbnb Data Analysis & Clustering

This project analyzes Airbnb listing data to uncover pricing, location, and availability patterns.
It applies exploratory data analysis and unsupervised machine learning techniques to segment listings
for potential personalization and optimization use cases.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Folium

## Key Work
- Data cleaning and preprocessing
- Feature engineering and scaling
- Dimensionality reduction using PCA
- Clustering using K-Means and DBSCAN
- Visualization using PCA, t-SNE, and geospatial maps
